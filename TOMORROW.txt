STEP 1 — ADD ATHLOS.FI AS CUSTOM DOMAIN IN VERCEL
--------------------------------------------------
1. Go to vercel.com → log in → open the Athlos project
2. Settings → Domains
3. Add "athlos.fi" and "www.athlos.fi"
4. Vercel will show DNS records to add (likely an A record and/or CNAME)
5. Copy these records — you need them for Step 2

Note: When athlos.fi is set as the production domain in Vercel,
the staging URL (athlos-website-pi.vercel.app) will automatically
redirect to athlos.fi. No extra config needed.


STEP 2 — UPDATE DNS AT GODADDY (or wherever DNS is managed)
------------------------------------------------------------
The old site runs on GoDaddy. DNS needs to point to Vercel instead.

1. Log in to GoDaddy DNS manager for athlos.fi
2. Remove or replace the old A record pointing to GoDaddy hosting
3. Add the A record and/or CNAME that Vercel provided in Step 1
4. Save changes
5. Propagation is usually fast (minutes) but can take up to 48 hours

To check propagation: visit athlos.fi and see if the new site loads.
Or use: https://dnschecker.org

STEP 4 — GOOGLE SEARCH CONSOLE (most important SEO action)
-----------------------------------------------------------
This is the single highest-impact action of the day.

1. Go to search.google.com/search-console
2. Add property → choose "Domain" type → enter: athlos.fi
3. Google will give you a DNS TXT record to verify ownership
4. Add that TXT record in GoDaddy DNS manager (same place as Step 2)
5. Click Verify in Search Console
6. Once verified: Sitemaps → Add sitemap → enter: https://athlos.fi/sitemap.xml
7. Submit

Then manually request indexing on the most important pages:
  - URL Inspection tool → paste each URL → Request Indexing
  Priority order:
    https://athlos.fi/
    https://athlos.fi/products/wios.html
    https://athlos.fi/products/ufs.html
    https://athlos.fi/products/ufs-ip67.html
    https://athlos.fi/about.html
