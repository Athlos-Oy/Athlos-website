Software & Integration

Software
Services

Decades of software expertise — available as licensable components or integrated with Athlos imaging systems.

Overview

The software layer behind direct conversion imaging.

Athlos has been developing X-ray imaging software since the early days of direct conversion — reconstruction algorithms, image processing filters, and TWAIN integration layers that have been deployed in thousands of clinical and industrial installations worldwide.

These software capabilities are available as part of a complete Athlos imaging system, or as independently licensable components for integration into third-party platforms. Athlos also provides SDK access and direct integration support for OEM and system integrator customers.

What's Available

Software components for imaging system development.

01

TWAIN Module

The Athlos TWAIN module integrates directly with dental and medical imaging software. Supported applications include Dentrix Ascend, Eaglesoft, Open Dental, Dexis, Romexis, and many more TWAIN-enabled platforms. Available as part of a hardware solution or as a standalone licensed component.

02

Reconstruction Algorithms

Advanced reconstruction algorithm libraries developed over decades of direct conversion imaging work. Designed for the specific signal characteristics of CdTe and Si CMOS detectors. Available as a standalone licensable component for integration into imaging system pipelines.

03

Post-Processing Filters

Locally adaptive image enhancement and post-processing filter libraries. Developed and refined across dental, industrial, and medical imaging applications. Provided with the Athlos SDK and available for integration and customisation.

Athlos SDK & API C/C++ integration API with source code examples, evaluation GUI, and advanced locally adaptive filter library. Contact Athlos to discuss SDK access and licensing.

Software licensing and integration questions.

Whether you are integrating Athlos software into an existing platform or evaluating it for a new imaging system, the Athlos team can support you. Contact us to discuss licensing, SDK access, and integration paths.
